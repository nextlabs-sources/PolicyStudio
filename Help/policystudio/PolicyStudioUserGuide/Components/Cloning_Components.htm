Cloning Components

For various reasons, you might need to create components that are very similar to existing ones, but with only a minor change. In such cases, it is more conve­nient to make a copy of the existing components and then make the desired changes in the copies, rather than build the entire component from scratch. This is referred to as cloning. All types of components can be cloned.

To clone components:

  1. Open the appropriate component bin and select the component(s) you wish to clone.

  2. Either right-click and select Duplicate, or open the Edit menu and select Duplicate. You can also use the Ctrl + D shortcut.

  3. In the Save In Folder field, select the folder location where you want the duplicate components to be saved. By default, the new copy is automati­cally assigned the name Copy of <OldName>. You have the option of assigning another name at this point, in the New Name field.

  4. Click Save to clone the selected components.

If the component is based on any other components (these are called leaf com­ponents), those will not be cloned; rather, the cloned copy and the source com­ponent will share the original leaves. If this does not suit your needs—that is, if you want to change the definition of the leaf components—you need to clone the leaf components as well, using the same procedure.

Policies and folders can also be cloned; see Cloning Policies.
