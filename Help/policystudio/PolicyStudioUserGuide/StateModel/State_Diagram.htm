State Diagram

The following figure represents the five object states and the possible transi­tions among them. Note that an object’s version number is incremented when­ever it passes into Deployed state, and at no other time. This occurs regardless of whether or not any changes have actually been made to the definition of the object. This means that technically there is nothing preventing different ver­sions of an object from having exactly the same definition.
