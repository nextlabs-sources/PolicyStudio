Welcome to 7.5

Welcome to NextLabs Control Center release 7.5, the leading enterprise-level platform for policy-driven data loss prevention and entitlement management. This user guide provides information on using Policy Studio, the application you use to define, deploy, monitor and manage your data loss prevention policies. It is intended for administrators and policy designers responsible for converting the general data access and usage management goals of the enterprise into deployable, active policies.

Information is broken into the following sections:

·        Introducing Policy Author

·        Defining Components

·        Constructing Policies

·        Introducing Policy Manager

·        How Do I...?

·        The Object State Model

·        Migrating 6.x Policies

What’s New in this Release

This release provides the following new features:

·        Capability to add tags to policies.You can use these tags later to search for specific policies in Reporter. For example, you can add a tag to indi­cate that a policy controls access to data classified as ITAR Export Con­trol. Later, in Reporter, you can define a query to view all policy activities where the policy tag is ITAR Export Control.

·        Capability to select multiple policy objects for Submit, Deactivate, and Delete operations.
