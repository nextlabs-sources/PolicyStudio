The  Administrator  Utility

The Entitlement Manager for SharePoint includes a command line utility that allows administrators to set certain configuration settings that control the Entitlement Manager’s content trimming features. This utility, called the SPE Administrator, is available on any SharePoint server where the Entitlement Manager is installed, and can be launched with the command

ce_spadmin.exe -o [feature] [flags]

For an overview of the functions of this utility, see the section, The SPE Administrator Utility.

As an alternative to controlling trimming features through SharePoint’s own administrative pages, you can use the special utility called Entitlement Manager for SharePoint Administrator. This utility is automatically installed on each Entitlement Manager for SharePoint host, in the C:\Program Files\NextLabs\SharePoint Enforcer\bin directory. It is available only to users with administrative permissions for the site collection, based on their Windows logon. The procedures for running the utility are the same for SharePoint 2007 and 2010.

The utility can be used to control each of the three kinds of security trimming described above.
